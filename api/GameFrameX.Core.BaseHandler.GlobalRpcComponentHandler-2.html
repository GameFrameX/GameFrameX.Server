Remarks

所有全局组件处理器均应继承自此抽象类，以实现统一的组件初始化、超时监控、性能统计与异常处理逻辑。 本泛型版本在单参版的基础上，额外提供强类型的组件代理访问能力，方便业务层直接操作具体组件代理而无需强制转换。
