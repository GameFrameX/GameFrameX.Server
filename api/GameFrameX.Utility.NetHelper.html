Remarks

核心功能：

  • IP地址格式验证和解析
  • 端口可用性检测和管理
  • 本地网络接口信息获取
  • 网络连接状态查询

适用场景：

  • 网络服务器端口分配
  • 客户端连接配置验证
  • 网络诊断和监控
  • 分布式系统节点发现
