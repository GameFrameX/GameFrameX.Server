表示异步操作的Task对象。当消息成功写入时完成,如果发生错误则抛出异常

Remarks

此方法用于将消息异步发送到网络通道。 如果errorCode不为0,接收方可以根据错误码进行相应的错误处理。 调用此方法时需要确保网络通道处于打开状态。
