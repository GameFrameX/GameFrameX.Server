Remarks

包含健康检查的完整响应数据，包括整体状态、各项检查详情、执行时间和服务器配置信息。 用于向客户端提供结构化的健康检查结果，便于监控和故障排查。

List containing all health check items / 包含所有健康检查项目的列表

Remarks

包含所有已注册的健康检查项目的详细信息，每个项目包含名称、状态、描述和执行时间。 用于提供完整的健康检查报告，便于详细分析系统各组件的健康状况。
