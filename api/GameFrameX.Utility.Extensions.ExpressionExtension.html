Not<T>(Expression<Func<T, bool>>)

对表达式进行逻辑非运算，对表达式的结果取反。

public static Expression<Func<T, bool>> Not<T>(this Expression<Func<T, bool>> expr)

Parameters

expr Expression<Func<T, bool>>

要进行逻辑非运算的表达式。

Returns

Expression<Func<T, bool>>

一个新的表达式，表示输入表达式的逻辑非运算结果。

Type Parameters

T

表达式的参数类型。

Remarks

如果输入表达式为 x => x > 5，则输出表达式为 x => !(x > 5)，等价于 x => x <= 5。

Exceptions

ArgumentNullException

当 expr 为 null 时抛出。
