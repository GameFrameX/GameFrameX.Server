IsImplWithInterface(Type, Type, bool)

判断类型是否实现了指定的接口。 此方法用于检查一个具体类型是否实现了目标接口。

public static bool IsImplWithInterface(this Type self, Type target, bool directOnly = false)

Parameters

self Type

要判断的类型。必须是非空的具体类型。

target Type

要判断的接口类型。必须是非空的接口类型。

directOnly bool

是否只检查直接实现的接口。 当设置为true时，只检查直接实现的接口； 当设置为false时，同时检查继承链上的所有接口。

Returns

bool

满足以下所有条件时返回true:

  1. self和target参数都不为null
  2. target是接口类型
  3. self不是接口类型或抽象类型
  4. self实现了target接口（根据directOnly参数决定检查范围） 否则返回false
