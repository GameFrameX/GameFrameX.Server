Remarks

提供用于配置和使用OpenTelemetry可观测性功能的扩展方法，支持指标收集、链路追踪和日志记录。 包含Grafana集成、Prometheus指标导出和独立指标服务器创建等功能。

CreateMetricsServerAsync(AppSetting, string)

Create standalone metrics server / 创建独立的指标服务器

public static Task<WebApplication> CreateMetricsServerAsync(AppSetting setting, string servicePrefix = "")

Parameters

setting AppSetting

Application settings / 应用设置

servicePrefix string

Service name prefix / 服务名前缀

Returns

Task<WebApplication>

Metrics server task / 指标服务器任务

Examples

// 创建独立指标服务器
var metricsServer = await OpenTelemetryExtensions.CreateMetricsServerAsync(appSetting);

// 带服务前缀
var metricsServer = await OpenTelemetryExtensions.CreateMetricsServerAsync(appSetting, "GameServer");

Remarks

创建一个独立的Web应用程序专门用于暴露Prometheus指标端点。 这样可以将指标收集与主应用程序分离，提高系统的可观测性和稳定性。 服务器会自动检查端口可用性，并在启动后输出访问地址。

Exceptions

ArgumentNullException

Thrown when setting is null / 当setting为null时抛出

InvalidOperationException

Thrown when metrics port is not available / 当指标端口不可用时抛出
