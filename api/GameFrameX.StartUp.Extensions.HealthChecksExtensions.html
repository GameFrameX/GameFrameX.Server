Examples

// 使用默认路径
app.UseGameFrameXHealthChecks(appSetting);

// 使用自定义路径
app.UseGameFrameXHealthChecks(appSetting, "/api/health");

Remarks

此方法会配置以下健康检查端点：

  • /health - 详细的JSON格式健康检查报告
  • /health/simple - 简单的"OK"响应（兼容性端点）
健康检查响应包含状态、检查项详情、持续时间、服务器信息和时间戳。
